Calculette Immo

Site vierge prêt à être publié sur GitHub Pages.

Une fois publié, soumettez /sitemap.xml à Google Search Console pour indexation.

Quoi faire ensuite

  1. Active GitHub Pages (Settings → Pages) sur la branche main, dossier /.
  2. Ajoute la balise de vérification Google Search Console (meta) puis soumets le sitemap.
  3. Quand le site est indexé, remplace cette page par ton application.

Fichiers fournis

  • index.html — page d'accueil minimale
  • 404.html — page 404
  • sitemap.xml — sitemap pour Search Console
  • robots.txt — permet l'indexation et pointe vers le sitemap
  • README.md — guide pour publier et référencer
  • .nojekyll — force Pages à servir les fichiers statiques
